Projects

   Mini Search Engine (Python)
Created a simple search engine using a web crawler to retrieve approximately 5,000 documents from the muhlenberg.edu domain. The documents were then indexed using an inverted index, which was leveraged when calculating Tf-Idf vector weights to generate a vector space model of all the documents and the user's query. Then, using cosine similarities, the highest ranked results were calculated and returned.
   HMM Part of Speech Tagger (Python)
Used the Viterbi algorithm to create a part of speech tagger using bigram calculations, hidden Markov model, and the NLTK Brown corpus to train and test the model. Achieved ~91% accuracy.
